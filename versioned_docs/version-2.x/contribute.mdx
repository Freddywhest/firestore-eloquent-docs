# Contribute

:::info[IMPORTANT]
This project is still in development and there are some features needed to be implemented. If you want to contribute, please read the [How to contribute](#how-to-contribute) section.
:::

## How to contribute

- Check the [todo list](/docs/todo) to see what features are needed to be implemented and what features are already implemented but need to be tested.
- Fork the repository or clone it.
- Create a new branch for each feature or improvement you're working on.
- Push your changes to your forked repository or your cloned repository.
- When you're done working on your feature or improvement, create a pull request from your branch to the `develop` branch of the original repository.
- After your pull request is merged, you can safely delete your branch.

Feel free to contribute by opening issues or submitting pull requests. Your feedback and contributions are highly appreciated :heart: :heart: :heart:
